# 竹签 · ZhuQian

一款用 **Rust** 与 **eframe / egui** 编写的桌面文本编辑工具，侧重写作与阅读体验，支持可定制的「标签」高亮（基于正则规则），不改变源文件中的普通文本内容。

> **说明**：本仓库是借助 **[Claude Code](https://claude.ai/code)** 进行开发与迭代的**实验性项目**，功能与接口可能随时调整，不适合作为稳定生产依赖。

![应用截图](screenshot.png)

## 功能概览

- **多标签编辑**：多文件标签页、未保存状态提示，切换标签时可配合自动保存流程使用。
- **多种格式**：`.txt`、`.md`、`.log`，以及专有格式 **`.zq`**（可在文件内嵌入显示偏好等元数据）。
- **高亮与主题**：用正则定义规则，设置前景/背景色、粗体等；规则可保存为**主题**（默认写入 `themes.json`）。
- **Markdown 渲染模式**：标题层级、粗体、斜体、行内代码、删除线等样式叠加在自定义高亮之上。
- **显示与预设**：字体、字号、文字颜色；编辑器与侧栏支持**纯色或背景图**；显示方案可存为预设（`display_presets.json`）。
- **侧栏**：可在**文件树**（目录浏览）与**大纲**（当前文档结构）等模式间切换，也可关闭侧栏。
- **文档统计**：底部展示行数、字符数、词数、标签数量等。

## 环境要求

- 已安装 [Rust](https://www.rust-lang.org/tools/install) 工具链（`rustc`、`cargo`）。

## 构建与运行

```bash
cargo build          # 调试构建
cargo build --release # 发布构建
cargo run             # 运行
```

## 快捷键

| 快捷键   | 作用           |
|----------|----------------|
| `Ctrl+S` | 保存当前文件   |

## 项目结构（简要）

| 路径 | 说明 |
|------|------|
| `src/main.rs` | 主界面、标签页、主题应用、字体与背景资源等 |
| `src/parser.rs` | 高亮规则、Markdown 布局、文档统计 |
| `themes.json` | 用户保存的高亮主题（运行后生成或更新） |
| `display_presets.json` | 显示预设（运行后可能生成） |

`.zq` 文件可在正文与元数据之间使用标记（如 `---zq-meta---` … `---end-meta---`）保存 JSON 形式的显示偏好，便于单文件携带个人排版设置。

## 平台说明

目标为跨平台桌面应用；开发时曾在 Windows 上加载系统字体目录等，其他系统上字体与路径行为以当前实现为准。
