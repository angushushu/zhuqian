竹签（zhuqian）是一款带有label高亮功能的文本编辑器。

## 核心功能

支持.md，.txt，.log等文本文件的编辑。同时支持.zq的专有文件格式。工具特点是支持自定义label高亮（不会改变源文件），比如默认会对`[]`中的内容高亮。同时支持不同等级的label，比如`[]`，`[[]]`，`[[[]]]`等，不同等级的label有不同的颜色。用户可以自定义不同类型的label的高亮规则，比如`()`，`(())`，`((()))`等，不同类型的label有不同的颜色。

## 界面

- [x] 界面分为左右两个部分，左侧是文件列表/目录（可选择切换或关闭），右侧是文本编辑器。
- [ ] 视觉效果应该是扁平化，类似于terminal/vim风格。
- [ ] 界面默从以下色彩选择：#E2DBAC，#AEC957，#6E9A42，#538D1A，#377C07，#2C6705
- [ ] logo使用zq_icon.png

## 进阶功能

显示设置和设置应该合并，并对应独立panel。可设置的包括：

- [x] 字体（默认times new roman）
- [x] 字体大小
- [x] 字体颜色
- [x] 每个规则对应的高亮颜色（规则使用正则表达式，颜色使用RBG/HEX设置,是否加粗）
- [x] 允许把高亮规则保存为主题
- [x] 面板背景颜色（纯色或图片）
- [x] 左侧可选显示文件目录（类似vscode）还是文件内目录（类似obsidian）
- [x] 允许关闭左侧目录面板
- [x] 下方显示文档信息（字数，行数，字符数，标签数量等）
- [ ] 需要支持theme删除
- [ ] 多语言支持
